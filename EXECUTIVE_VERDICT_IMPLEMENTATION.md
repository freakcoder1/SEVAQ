## 🚨 CRITICAL ISSUES TO FIX

### 1️⃣ TRUST HEADER - STILL WRONG (SUBTLE BUT CRITICAL)

**Current Problem:**
```
Lat: 28.7040592, Lng: 77.1024901999
```

**Why This Is Bad:**
- Coordinates feel engineering-first
- Breaks emotional safety
- Reminds users they are "inside a system"
- No consumer app should show raw coordinates

**Required Fix:**
```
Primary line: Sector 62, Noida
Secondary line: We're monitoring availability in your area
If unknown: Confirming your service area...
```

### 2️⃣ HERO CARD - GOOD, BUT TOO EMPTY

**Current Problem:**
Feels emotionless, like "Here is a service. Click if you want."

**Required Fix:**
Add ONE human reassurance line:
- "Handled daily by our team"
- "Trusted by homes nearby"
- "We'll assign the right professional"

### 3️⃣ CTA COPY - 90% RIGHT, 10% OFF

**Current:** "We'll handle this"
**Better:** "We'll take care of this" or "Yes, handle this for me"

### 4️⃣ EMPTY SPACE PROBLEM

**Current:** Screen feels too empty below hero card
**Fix:** Add one muted divider section:
```
How Sevaq works
We assign, monitor, and replace if needed — automatically.
```

### 5️⃣ SUPPORT SIGNAL - GOOD IDEA, BAD PLACEMENT

**Current:** Too far down, feels like footer text
**Fix:** Move closer to hero card, reduce opacity to 70%
